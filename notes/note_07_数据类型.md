## 数据类型

Python中数据类型可以分为数字型和非数字型

### 数字型

- 整型 `int`
- 浮点型 `float`
- 布尔型 `bool`
  
        真 True 非0--非零即为真
        假 False 0

- 复数型 `complex`

### 非数字型

- 字符串
- 列表
- 元组
- 字典

-------------

### 在Python中，所有非数字型变量都支持以下特点

1. 都是一个序列`sequence`，也可以理解为容器
2. 取值`[]`
3. 遍历 `for in`
4. 计算长度、最大值/最小值、比较、删除
5. 链接 `+` 重复 `*`
6. 切片
